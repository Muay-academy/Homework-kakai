``` java / javac filename```

2. PHP

``` php filename.php```

3. Python

``` pyjd hello.py ```

``` npm run, yarn run ```

5. Typescript

``` tsc filename.ts  -> node filename.ts ```

6. Ruby

``` ruby filename.rb ```

7. go

``` go run dir```

8. webAssembly

```wasmtime --dir . build/index.wasm```

9. rust

``` cargo build -> cargo run ```

10. node.js

``` npm run command ```

11. Flutter

``` flutter run  lib/main.dart```

12. bun

``` bun run command```

13. C#

``` csc filename.cs```

14. Carbon

``` brazel run filename.carbon```

15. deno

```deno run filename.ts```
